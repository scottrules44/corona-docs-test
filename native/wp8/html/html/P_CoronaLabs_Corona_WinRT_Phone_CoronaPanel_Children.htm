Collection of UI elements/controls to be displayed within the Corona panel's bounds.
Remarks
The owner of this Corona panel control can use this property to add Xaml controls within Corona control. In the Visual Studio UI designer, dragging and dropping Xaml control within this Corona control causes them to be added to this property's collection.
