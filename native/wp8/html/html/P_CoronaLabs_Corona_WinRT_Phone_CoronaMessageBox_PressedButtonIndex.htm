Indicates which button was pressed to close the message box.

Note that this property is only applicable if the WasButtonPressed property is set true.

public int PressedButtonIndex { get; }
Public ReadOnly Property PressedButtonIndex As Integer
	Get

Property Value

Type: Int32

Zero based index of the button that was pressed to close the message box. This index matches a button label in the ButtonLabels collection.

Set to -1 if a button was not pressed to close the message box, which is indicated by the WasButtonPressed property. In this case, the message box was closed via the back key or by one of the Close() methods.
