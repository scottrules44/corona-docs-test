	Get

Property Value

Type: IListString

Modifiable collection of button labels to be displayed as button in the message box.

Can be empty, which means that the message box will not display any buttons and can only be closed by the end-user via the back key.

Remarks
You are expected to set this property before calling the Show method. Any changes made to this property while the message box is shown will not be displayed until the next time the message box is shown.
