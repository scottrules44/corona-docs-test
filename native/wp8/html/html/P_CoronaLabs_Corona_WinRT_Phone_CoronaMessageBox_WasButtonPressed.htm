public bool WasButtonPressed { get; }
Public ReadOnly Property WasButtonPressed As Boolean
	Get

Property Value

Type: Boolean

Set true if the message box was closed via a button press, in which case, property PressedButtonIndex will indicate which button was pressed. Note that this will be true if you call the CloseWithButtonIndex(Int32) method as well.

Set false if the message box was closed via the back key, if a Close() method was called, or if the message box hasn't been shown yet.
